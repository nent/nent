What is it?

nent is a set of custom HTML elements (web components) to enrich static HTML files with application features. Our focus is on creating experiential websites with less scripting and no build-process.

What it isn't.

nent functional elements add functionality, NOT design. These elements work with your design. Since it is HTML, you can always use it with any framework, element library or UI kit you want.

Low-Code, No Framework Needed

The elements bring application functionality to HTML with one script reference and no code or build-process required.
