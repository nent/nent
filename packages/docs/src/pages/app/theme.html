System theme detection and selection.

Native Dark Mode

NENT comes with an element that detects and applies the user's theme preference. Additionally, there's a toggle control you can use to give the user override control. Once they've used the switch, the preference is stored in session.

You can also add your own icons or control to change the style of your dark-mode switch.
