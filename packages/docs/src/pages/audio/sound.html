Sounds

This page has on-demand sounds configured to play when buttons are clicked.

Add sound-effects or voice-overs declaratively

Sounds are a little different than music. We don't loop sounds and they need to play on-demand. There are ways to trigger them, but this page attaches them to a button.

First: Preload

To get the timing perfect, we have to pre-load all sounds for a route using the <n-audio-action-sound-load> element. The default mode for sounds is load. This gets the audio ready to play and waits for the start command.

Then: Activate

We play the sound by sending an action with the start command. We can use the <n-action-activator> element and special <n-audio-action-sound> elements with the command, and track-id.
