Child Routes

Routes, child-routes, their child-routes ad-infinitum.

A route is a path segment in the URL. A child-route is a new path segment under that route:

/route/child-route

How it Works

The first-level must be a DIRECT child of the <n-views> element however, any declared deeper in the HTML will create child-routes.

Nested routes are declared statically using the <n-view> (and <n-view-prompt>) elements.
