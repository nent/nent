How it Works

Regular anchor tags (<a>) inside the <n-views> tag will have their click intercepted for fast, in-place content replacement without refreshing the entire page.

Automatic Links

Nent can automatically create links to the <n-view-link-next > view, or <n-view-link-back > to the previous view using special elements. Also, you can create menus, breadcrumbs and even progress maps or tabs using the <n-view-link-list > element.
